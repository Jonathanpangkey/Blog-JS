# Blog-JS
The Blog App is a web application built using Express and MongoDB. It allows users to create, read, update, and delete blog posts, as well as leave comments on each blog post.
Features :

    User Authentication: Users can register, log in, and log out to access the application.
    Flash Messaging: Provides feedback messages to users for successful actions and errors.
    CRUD Operations: Users can create, read, update, and delete blog posts.
    Commenting: Users can leave comments on each blog post.
